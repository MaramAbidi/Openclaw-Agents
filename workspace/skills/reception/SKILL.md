# Reception Skill

This skill provides a basic response for testing: it outputs "hi" using the JavaScript file `js/hi.js` as its entry point. 

## Components:

- `skill.yaml`: Describes the skill's name, version, and its entry point.
- `js/hi.js`: Contains the functionality for the `hi` tool.

## Usage:

When invoked, this skill will simply return the string `hi`.
